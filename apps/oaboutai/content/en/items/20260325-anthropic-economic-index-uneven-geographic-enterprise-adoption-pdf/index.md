## Detailed Notes

- **Publication identity:** The document is explicitly titled “Uneven geographic and enterprise AI adoption” and published on 2025-09-15.
- **Core thesis:** AI adoption is fast, but concentration across places and use cases resembles early-stage diffusion seen in prior general-purpose technologies.
- **Time-series usage tracking:** The report compares Claude.ai usage across multiple prior windows to detect directional changes in task composition.
- **Rising education/science share:** It reports growth in educational and scientific task categories while coding remains dominant.
- **Shift toward directive automation:** Directive interactions increased materially over time, signaling greater end-to-end delegation in some workflows.
- **Geographic index contribution:** Introduces AUI to compare observed usage vs expected usage by working-age population.
- **Cross-country inequality pattern:** High-income countries are disproportionately represented in per-capita usage.
- **U.S. local-economy effects:** State-level usage differences align with local industry composition and occupational mix.
- **Diversity of use in high-adoption regions:** Higher-adoption geographies show broader task distribution, not only coding-centric usage.
- **API vs consumer divergence:** First-party API traffic is more automation-heavy than Claude.ai interactive usage.
- **Enterprise specialization:** API usage skews toward coding and office/admin domains compared with broader consumer educational/writing usage.
- **Capability over cost signal:** The report finds weak price sensitivity for common enterprise tasks relative to capability fit and economic value.
- **Context bottleneck finding:** High-impact enterprise deployment is constrained by data/context readiness and organizational modernization.
- **Diffusion-risk implication:** Without intervention, AI productivity gains may concentrate in already-advantaged regions and firms.
- **Research utility:** Open data release is intended to support independent labor-market, adoption, and policy-impact studies.
- **Strategic takeaway:** Sustainable AI policy should target diffusion quality (skills, context, infrastructure), not only raw access.
